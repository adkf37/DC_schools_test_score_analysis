# Input Data

Place your DC school test score Excel files here.

## Expected Files:
- `2021-22 School Level PARCC and MSAA Data.xlsx`
- `2022-23 School Level PARCC and MSAA Data_9_5.xlsx`
- `DC Cape Scores 2023-2024.xlsx`
- `2024-25 Public File School Level DCCAPE and MSAA Data 1.xlsx`

## Optional:
- `school_locations.csv` - For map visualization in the Dash app
  - Required columns: `School Name`, `Latitude`, `Longitude`

**Note**: Data files are not tracked by git (see `.gitignore`)
